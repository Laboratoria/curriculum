# Clase de Conceptos
- Tipo: `clase presencial`
- Formato: `guided-learning`
- Duración: `120min`

## Objetivos

- Revisar y consolidar los conceptos aprendidos por medio de una clase con tu equipo de profesores
- ¡Aclarar todas las dudas que tengas!
