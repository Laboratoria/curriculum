# findPairForSum

Dada una lista de números enteros no negativos y una suma objetivo, encuentra un
par de números que sumen a la suma objetivo.

Ejemplo de entrada:

```js
const pair = findPairForSum([3, 34, 4, 12, 5, 2], 9);
console.log(pair); // --> [4, 5]

```
